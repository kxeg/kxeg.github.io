匿名函数

发表于 2016-08-12

所谓的匿名函数，就是不需要显式的定义函数，有的时候，使用匿名函数更加方便。
在python中，关键字lambda表示匿名函数，以map（）函数举个例子，计算f(x)=x3时，可以这样子

>>> list(map(lambda x: x**3,range(1,5)))
[1, 8, 27, 64]

实际上，lambda x: x**3就等同于

def f(x):
    return x * * 3

匿名函数有个限制，就是只能有一个表达式，不用写return，返回值就是该表达式的结果。
用匿名函数有个好处，因为函数没有名字，不必担心函数名冲突。此外，匿名函数也是一个函数对象，也可以把匿名函数赋值给一个变量，再利用变量来调用该函数

>>> f = lambda x,y:x+y
>>> f
<function <lambda> at 0x000001E7B4637F28>
>>> f(5,5)
10

匿名函数作为返回值返回，比如：

def sum(x,y):
    return lambda : x + y

需要提醒一下的是，这里的匿名函数作为返回值返回相当于

def sum(x,y):
    def g():
        return x + y
    return g

如果你直接使用sum(1,2），那么它返回的将是g函数，所以你必须使用f = sum(1,2),然后f()才能输出3这个数字哦。

Python对匿名函数的支持有限，只有一些简单的情况下可以使用匿名函数
